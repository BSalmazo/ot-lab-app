# TRP Input Package (For ChatGPT)

This folder is a thesis-oriented package built from the validated checkpoint campaign and subsequent technical discussions about real OT deployment constraints.

## Goal

Use this package to draft a **Thesis Research Plan (TRP)** in English, aligned with institutional guidance:

- context, motivation, and objectives,
- early thesis statement,
- state of the art and research gap,
- preliminary work already completed,
- commented workplan with timeline/Gantt.

## Recommended files to provide first

1. `TRP_MASTER_BRIEF.md`
2. `TRP_RESEARCH_CONTEXT_AND_OBJECTIVES.md`
3. `TRP_STATE_OF_ART_AND_GAP_MAP.md`
4. `TRP_PRELIMINARY_WORK_AND_VALIDATION.md`
5. `TRP_DEPLOYMENT_ARCHITECTURE_OT.md`
6. `TRP_WORKPLAN_GANTT.md`
7. `TRP_WRITING_PROMPT.md`
8. `TRP_DATA_MANIFEST.json`

## Important writing policy

- Conversation can be in Portuguese, but TRP text and all exported artifacts must remain in English.
- Do not invent numerical results or deployment claims not supported by the checkpoint evidence.
- Explicitly separate what is already validated from what is future work.
