macOS

Intel / Apple Silicon (Universal)

⬇️ Download Agent 📋 Install Script

Quick Start

  1. Download both the agent and install script
  2. Open Terminal in Downloads folder: cd ~/Downloads
  3. Run the install script: bash install-macos.sh
  4. Start the agent: ./otlab-agent-macos-amd64

Requirements

  • macOS 11.0 or later
  • libpcap (usually pre-installed)
  • If missing: brew install libpcap
Still seeing "cannot verify" warning?

After running the script, go to System Settings → Privacy & Security and click "Open Anyway"

Windows

64-bit Executable

⬇️ Download Agent 📋 Install Script

Quick Start

  1. Download both the agent and install script
  2. Run the install script:
    • Right-click install-windows.bat
    • Select "Run as Administrator"
  3. Double-click the agent to run it

Requirements

  • Windows 10 or later (64-bit)
  • Npcap driver (required)
  • Download Npcap from nmap.org →
  • Choose "Npcap Installer" (not WinPcap)
SmartScreen Warning?

The install script removes this automatically. If it still appears, click "More info" → "Run anyway"

Linux

64-bit Executable

⬇️ Download Agent 📋 Install Script

Quick Start

  1. Download both the agent and install script
  2. Open terminal in Downloads folder: cd ~/Downloads
  3. Run the install script: bash install-linux.sh
  4. Start the agent: ./otlab-agent-linux-amd64

Requirements

  • Ubuntu 20.04+, Debian 10+, CentOS 8+, or compatible
  • libpcap (required)
  • Ubuntu/Debian: sudo apt-get install libpcap0.8
  • CentOS/RHEL: sudo yum install libpcap
  • Arch: sudo pacman -S libpcap
libpcap installation automatic

The install script detects your package manager and shows installation commands if needed

Common Issues

  • Q: "Permission Denied" on macOS/Linux?

    A: Try: chmod +x install-*.sh

  • Q: How do I check if Npcap is installed (Windows)?

    A: Run: devcon status npcap in Command Prompt

  • Q: Can I run the agent without Npcap/libpcap?

    A: No, it will exit with clear error message showing how to install
