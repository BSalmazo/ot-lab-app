Download the Local Runtime package for your operating system. It provides packet capture, Modbus services, and process PLC/HMI control for the web dashboard.

Setup Notes

  • Use the package download first; it is generated from the GitHub release asset that matches the deployed web build.
  • Run the included install script before starting the Local Runtime.
  • The generated agent-config.json binds the runtime to your current web session.
